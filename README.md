## Security Quick Reference

| Threat | First Action |
|--------|-------------|
| Suspicious email received | Do not click links or open attachments; report to security team |
| Clicked a suspicious link | Close browser immediately, change password, run malware scan, report |
| Entered credentials on a fake site | Change password immediately from a safe device, enable MFA, report |
| Account locked out unexpectedly | Contact IT Service Desk through a verified channel (phone or in person) |
| Received MFA prompt you did not initiate | Deny the prompt, change your password immediately, report to security |
| Files encrypted or missing | Disconnect from network immediately, do not pay ransom, report to security |

## Reporting Suspicious Activity

| Method | How To Access |
|--------|---------------|
| Report phishing in Outlook | Select email > Report Message > Phishing |
| Report phishing in Gmail | Open email > three dots > Report phishing |
| Forward phishing as attachment | Open email > three dots > Forward as Attachment |
| Contact IT Security Team | Use your organization's official security contact |

---

For internal use. Report all security concerns to your organization's security team immediately.
