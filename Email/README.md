## Outlook Shortcuts Quick Reference

| Action | How To Access |
|--------|---------------|
| Force Send/Receive | F9 or Send/Receive tab > Send/Receive All Folders |
| Toggle Work Offline | Send/Receive tab > Work Offline |
| Open Account Settings | File > Account Settings > Account Settings |
| Manage Rules & Alerts | File > Manage Rules & Alerts |
| Mailbox Cleanup | File > Tools > Mailbox Cleanup |
| Open Mail control panel | Control Panel > Mail (Microsoft Outlook) |
