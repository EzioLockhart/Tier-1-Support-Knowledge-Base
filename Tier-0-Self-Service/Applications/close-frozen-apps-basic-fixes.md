## Fix 1: Close a Frozen Application

When an application freezes, the X button and clicking do nothing. Here is how to force it closed.

**Method A: Use Task Manager (Fastest)**
1. Press **Ctrl + Shift + Esc** on your keyboard — all three keys at the same time
2. Task Manager will open
3. If you see a small window, click **"More details"** at the bottom
4. Find the frozen application in the list (it may say "Not Responding" next to it)
5. Click on the application name to select it
6. Click the **"End task"** button at the bottom right
7. The application should close within a few seconds
8. Reopen the application normally

**Method B: Use Alt + F4 (Quick Close)**
1. Click on the frozen application window to make sure it is selected
2. Press **Alt + F4** on your keyboard — hold Alt and press F4
3. This sends a close command to the active window
4. If the application is completely frozen, this may not work — use Method A instead

**IMPORTANT:** Force closing an application will lose any unsaved work. Try waiting 2 minutes first — sometimes the application recovers on its own.

## Fix 2: Application Will Not Open At All

1. **Restart your computer first:**
   - Click the Start button (Windows icon)
   - Click the Power button
   - Select **"Restart"**
   - Wait for the computer to restart and try opening the application again
2. **Try opening as Administrator:**
   - Find the application in the Start menu
   - Right-click the application
   - Select **"More"** → **"Run as administrator"**
   - Click "Yes" if a permission prompt appears
3. **Check for updates:**
   - Open the application if possible
   - Look for a "Check for Updates" option (usually under Help, Settings, or About)
   - Install any available updates and restart the application

**Why restart helps:** Restarting clears temporary files and stuck background processes that may be blocking the application from opening.

## Fix 3: Application Is Very Slow

1. **Close other applications you are not using:**
   - Look at your taskbar at the bottom of the screen
   - Close any programs you do not need right now
   - Each open program uses memory
2. **Close browser tabs:**
   - If you have many browser tabs open (especially 10+), close the ones you are not using
   - Each tab uses memory and processing power
3. **Save your work and restart the application:**
   - Close the slow application completely
   - Wait 10 seconds
   - Reopen it
4. **Restart your computer:**
   - If the application is still slow after restarting it, restart the whole computer
   - A full restart clears everything and gives you a fresh start

**Why this helps:** Applications need memory (RAM) to run. If too many programs are open, there is not enough memory for each one, and everything slows down.

## Fix 4: Application Keeps Crashing

1. **Update the application:**
   - Open the application
   - Look for "Check for Updates" in the Help menu or Settings
   - Install any updates
2. **Repair the application (Windows built-in tool):**
   - Click Start > Settings (gear icon) > Apps > Installed apps
   - Find the crashing application in the list
   - Click the three dots (...) next to it
   - Select **"Modify"** (if available)
   - Choose **"Repair"** and follow the steps
3. **Reinstall the application:**
   - If repair does not work, uninstall and reinstall
   - Go to Settings > Apps > Installed apps
   - Find the application, click three dots, select **"Uninstall"**
   - Download the latest version from the official website
   - Install fresh and restart your computer

**Before reinstalling:** Make sure you know your login credentials and any license keys, as reinstalling may require them.

## Fix 5: Application Looks Strange or Broken

1. **Check the display scaling:**
   - Right-click on the desktop and select "Display settings"
   - Under "Scale & layout", ensure scaling is set to the recommended value
   - If text or windows look too large or small, this is often the cause
2. **Run in compatibility mode (older applications):**
   - Right-click the application shortcut
   - Select "Properties"
   - Go to the "Compatibility" tab
   - Check "Run this program in compatibility mode for:"
   - Select "Windows 8" or "Windows 7" from the dropdown
   - Click OK and try opening the application again
3. **Update your graphics driver:**
   - This is more advanced — if the above steps do not work, contact IT support

## When to Call the Helpdesk

Contact IT support if:

- The same application keeps freezing every day
- You cannot open an application that you need for work
- Task Manager does not open or the computer is completely frozen
- The application is business-critical and you need it working immediately
- You tried reinstalling and the problem persists

---

## Prevention Tips

- Save your work frequently — press Ctrl + S as a habit
- Close applications you are not using to free up memory
- Restart your computer at least once a week
- Keep your applications updated — updates fix bugs that cause freezing
- Do not open too many applications or browser tabs at once
