## Section A: How to Install a Program

### Step 1: Download from a Trusted Source

Only download software from official websites. This protects you from viruses and malware.

**How to find the official website:**
1. Search for the program name in your web browser
2. Look for the official website (usually the program name followed by .com, .org, or .net)
3. Avoid sites like "download.com", "softonic", or "cnet" — these often add unwanted extras
4. Look for a **"Download"** button on the official website

**Examples of trusted sources:**
- Microsoft Teams: teams.microsoft.com
- Zoom: zoom.us/download
- Adobe Reader: get.adobe.com/reader
- Google Chrome: google.com/chrome

### Step 2: Run the Installer

1. Once downloaded, open the installer file (usually in your Downloads folder)
2. The file is typically named `setup.exe`, `installer.exe`, or ends with `.msi`
3. Double-click the file to run it

### Step 3: Follow the Installation Prompts

1. If you see "Do you want to allow this app to make changes?" — click **"Yes"**
2. Follow the on-screen instructions
3. You can usually click **"Next"**, **"Install"**, or **"Continue"** to accept the default settings
4. Wait for the installation to complete (this may take a few minutes)
5. Click **"Finish"** when done

### Step 4: Find Your New Program

1. Click the **Start button** (Windows icon)
2. Scroll through the list of programs or type the name to search
3. Click the program to open it

## Section B: How to Uninstall a Program

### Step 1: Open the Installed Apps List

1. Click **Start** > **Settings** (gear icon)
2. Click **"Apps"** > **"Installed apps"** (Windows 11) or **"Apps & features"** (Windows 10)
3. You will see a list of all programs on your computer

### Step 2: Find the Program

1. Scroll through the list or use the search bar
2. You can sort by **"Size"** to see which programs take up the most space

### Step 3: Uninstall

1. Click the **three dots (...)** next to the program name
2. Select **"Uninstall"**
3. Click **"Uninstall"** again to confirm
4. Follow any prompts from the uninstaller
5. Wait for the process to complete

### Step 4: Restart If Needed

Some programs ask you to restart your computer. If prompted, save your work and click **"Restart"**.

## Fix 1: Program Won't Install

If an installer fails or freezes:

1. **Restart your computer** — this clears temporary issues
2. **Run as administrator:** Right-click the installer file > **"Run as administrator"**
3. **Check free disk space:** You need space for the program. Check C: drive in File Explorer
4. **Download a fresh copy:** The installer file may be corrupted. Delete it and download again
5. **Check if an older version exists:** Uninstall any previous version first (Section B)
6. **Temporarily disable antivirus:** Some antivirus programs block installations. Turn it off briefly, install, then turn it back on

---

## Fix 2: Program Won't Uninstall

If a program refuses to uninstall or the uninstall fails:

1. **Restart your computer** — sometimes programs are locked because they are running
2. **Close the program completely:** Check the system tray (bottom-right) for hidden icons
3. **Use the Control Panel method:**
   - Open Control Panel > Programs > Programs and Features
   - Find the program > right-click > Uninstall
4. **Reinstall then uninstall:** Sometimes installing the program again fixes the broken uninstaller, then you can uninstall properly
5. **Contact IT support** if the program is business-critical or the uninstall fails repeatedly

## Tip: See What Is Installed on Your Computer

To see a complete list of installed programs:

1. Click Start > **Settings** > **Apps** > **Installed apps**
2. Sort by **"Install date"** to see recently added programs
3. Sort by **"Size"** to see which programs use the most space
4. Look for programs you do not recognize or use — these are candidates for removal

## Safety Tips for Installing Software

- Only download from official websites — never from pop-up ads or email links
- Read each screen before clicking "Next" — some installers try to add extra programs
- Uncheck boxes for "additional offers" or "toolbars" during installation
- If the installer offers "Express" or "Custom" installation, choose "Custom" to see what is being installed
- Never download programs that claim to "speed up your computer" or "clean your registry" — these are often scams

## When to Call the Helpdesk

Contact IT support if:

- You cannot install a program you need for work
- The installer asks for an administrator password you do not have
- You are unsure if a program is safe to install
- A program will not uninstall after trying all methods
- The program is required by your organization and must be installed by IT
- You see error messages during installation or uninstall

## FAQ

**Q: Is it safe to delete programs I do not use?**
**A:** Yes, for most programs. Do not uninstall anything from Microsoft that you do not recognize (these may be system components). If you are unsure, search the program name online or ask IT.

**Q: What is the difference between "Uninstall" and "Delete"?**
**A:** Uninstalling removes the program properly, including its files and settings. Simply deleting the program folder or shortcut does not fully remove it. Always use the Uninstall option in Settings.

**Q: Why does installing software ask for administrator permission?**
**A:** Most programs need to write files to protected folders like Program Files. The permission prompt (User Account Control) is a security feature — it makes sure you want to allow the installation.
