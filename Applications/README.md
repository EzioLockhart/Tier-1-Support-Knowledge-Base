## Keyboard Shortcuts and Event Viewer Quick Reference

| Action | How To Access |
|--------|---------------|
| Open Task Manager | Ctrl + Shift + Esc |
| Close frozen application | Alt + F4 (on the frozen window) |
| Run as administrator | Right-click application > Run as administrator |
| Open Event Viewer | Win + R > eventvwr.msc |
| Application crash event ID | Event ID 1000 (Application Error) |
| Windows Error Reporting | Event ID 1001 |
| Performance Monitor | Win + R > perfmon |
